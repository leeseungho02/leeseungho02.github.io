알고리즘-백준

A - B

난이도: 브론즈 V

Feb 22, 2023
1 분 정도

원글

https://www.acmicpc.net/problem/1001

문제

두 정수 A와 B를 입력받은 다음, A-B를 출력하는 프로그램을 작성하시오.

입력

첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

출력

첫째 줄에 A-B를 출력한다.

예제

예제 입력 1

3 2

예제 출력 1

1

    // 출력
    cout << (A-B);
    
    return 0;
}
